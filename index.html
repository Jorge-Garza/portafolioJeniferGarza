Portafolio (Proyectos Destacados)

Introducción

Dentro del plan de estudio de la carrera de diseño industrial se encuentran diferentes meterias prácticas en las que podemos desarrollar nuestras habilidades como diseñadores, nos permite representar nuestras ideas en papel para después llevarlas a cabo, creando una variedad de productos con materiales diferentes, combinando texturas y acabados para obtener un producto interesante e innovador. ​

Dibujo.

Ésta materia es una de las más importantes de la carrera, pues nos ayuda a plasmar y comunicar nuestras ideas de manera rápida y eficiente sin necesidad de utilizar instrumentos.

Taller de Maquetas

Ésta materia nos perimte respresentar los bocetos en 3D, utilizando diversos materiales con los que podemos darle diferentes acabados y texturas para hacerlos más interesantes.

Diseño Básico

Ésta materia es crucial para tener un buen desempeño dentro de la carrera, pues nos permite aprender y aplicar los diferentes conceptos necesarios que se requieren para crear un buen producto.

Geometría Descriptiva

En ésta materia, a diferencia de dibujo, se requiere una mayor precisión a la hora de trazar, por lo que se solicita el uso necesario de instrumentos que nos ayudarán a trazar de forma correcta nuestros dibujos.

Objetos en perspectiva.

TALLER DE MAQUETAS

Texturas

En este trabajo se pueden observar distintos tipos de acabados que se pueden lograr en el material "gris pardo".

Instrumento

Bajo eléctrico relizado con el material "cartón corrugado", se utilizó gris pardo para algunos detalles y se le dieron acabados con pintura y barniz para un mejor resultado. Es una réplica a escala real.

Cámara Nikon F

Réplica de tamaño real de la cámara Nikon F, se realizó con el material "gris pardo" y se le dieron distintos acabados con pintura para recrear las diferentes texturas.

DISEÑO BÁSICO

Plano seriado

Para hacerlo se cortaron alrededor de 25 piezas de gris pardo del mismo tamaño para crear planos que se van uniendo para representar una composición. Se puso en práctica también el tema de "teoria del color" al elegir una combinación adecuada de colores.

Repentina: Portavasos

Este fue un proyecto realizado durante la "repentina" se trata de una serie de portavasos relizados con gris pardo, los cuales tienen un acabado de mosaico hecho con el mismo material.

Perchero.

Este trabajo se llevo a cabo por medio de una composición realizada anteriormente, en la que se expresa la emoción del ENOJO, por lo cual se utilizó una combinación de colores y de formas para representarlo de manera adecuada. Está hecho con gris pardo y se le dieron acabados con pintura y resanador.

Isometrías

Para realizar estas isometrías primero se obtuvieron las vistas de ambas piezas para tener las dismeniones correctas y trazarlas adecuadamente.

Vistas ortogonales y dimensionamiento

Se realizan para visualizar correctamente las caras de un objeto en diferntes ángulos, así como tambien para colocar las dimensiones correctas de los mismos.
